Progressive Web Apps

Basics

  • Served over https...must be secure. You are storing files on somebody else's computer. Scary!
  • Service workers...sites work quickly on any connection; fast, slow, non-existant.
  • Web manifests...add site to home screen on Android phone; add theme, splash screen with icon, eliminate browser "chrome." An App without the App store!

Checklist

from Google Developer's Web Fundamentals
  • Served over https
  • Responsive on tablets and mobile devices
  • App URLs load while offline
  • Meta data provided for "Add to home screen"
  • First load fast even on 3g
  • Site works cross-browser
  • Page transitions don't feel like they block on network
  • Each page has a URL
